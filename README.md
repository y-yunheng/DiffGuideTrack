# ODtrack: Object Detection and Tracking Framework

This repository contains an advanced object detection and tracking framework built upon transformer-based architectures. It includes various model configurations, training scripts, and evaluation tools for visual object tracking tasks.

## Features
- Transformer-based tracking architecture
- Advanced fusion modules for feature integration
- Support for LaSOT-format datasets (Anti-UAV410, CST-AntiUAV)
- Multi-GPU training capabilities
- Comprehensive evaluation tools
